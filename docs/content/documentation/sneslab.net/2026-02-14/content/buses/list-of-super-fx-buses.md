The following buses exist on the Super FX:

- HA Bus - Host Address
- HD Bus - Host Data
- X Bus
- Y Bus
- Z Bus
- PC Bus - program counter
- RAM A Bus - a 20-bit address bus
- ROM D Bus - an 8-bit data bus
- RAMD\_IN Bus - an 8-bit read data bus
- RAMD\_OUT Bus - an 8-bit write data bus
