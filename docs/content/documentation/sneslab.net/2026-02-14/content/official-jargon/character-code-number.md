The **Character Code Number** is Nintendo's terminology for "tile id." It is a 10-bit unsigned integer that indexes into a tilemap. It can range from 000h ~ 3FFh.

- [Appendix A-10 of Book I](https://archive.org/details/SNESDevManual/book1/page/n204) of the official Super Nintendo development manual
