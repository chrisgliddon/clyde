**Screen Over** is the name of the most significant half-nibble of M7SEL.

When 00, it indicates screen repetition should be used outside the screen area.

When 10, it indicates the backdrop color should be used outside the screen area.

When 11, it indicates character #0 repetition should be used outside the screen area.
