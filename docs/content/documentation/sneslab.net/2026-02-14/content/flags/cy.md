**CY** (CarrY) is the Super FX's carry flag. It is affected by the following instructions:

- ADC Rn
- ADC #n
- ADD Rn
- ADD #n
- ASR
- CMP Rn
- DIV2
- FMULT
- LMULT
- LSR
- MERGE
- ROL
- ROR
- SBC Rn
- SUB Rn
- SUB #n
