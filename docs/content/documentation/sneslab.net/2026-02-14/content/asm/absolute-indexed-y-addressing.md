**Absolute Indexed, Y Addressing** is supported by nine instructions:

- ADC (opcode 79)
- AND (opcode 39)
- CMP (opcode D9)
- EOR (opcode 59)
- LDA (opcode B9)
- LDX (opcode BE)
- ORA (opcode 19)
- SBC (opcode F9)
- STA (opcode 99)

#### Syntax

```
LDA addr, Y
```
