**Direct Page Indexed Indirect, X Addressing** is supported by eight instructions:

- ADC (opcode 61)
- AND (opcode 21)
- CMP (opcode C1)
- EOR (opcode 41)
- LDA (opcode A1)
- ORA (opcode 01)
- SBC (opcode E1)
- STA (opcode 81)

#### Syntax

```
LDA (dp, X)
```
