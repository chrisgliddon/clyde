Curated snapshot of [sneslab.net](https://sneslab.net) wiki. Game-specific content, ROM hacking, accessories, coprocessors (SA-1, Super FX, DSP-1), and stub articles have been removed. ~690 files retained covering core 65816/SNES development reference.

## Sections

- **asm/** — 65816 + SPC700 instruction reference (opcode, flags, timing)
- **snes-hardware/** — Hardware architecture, memory, chips
- **video/** — Graphics modes, VRAM, tile formats, sprite evaluation
- **audio/** — BRR, ADSR, DSP registers, FIR filter, Gaussian interpolation
- **registers/** — CPU register documentation
- **flags/** — CPU and hardware flag bits
- **addressing-modes/** — 65816 addressing mode details
- **map-modes/** — Background mode specs
- **buses/** — Address/data bus architecture
- **tables/** — ADSR timing tables, hardware tables
- **ram-maps/** — WRAM layout
- **aram-maps/** — Audio RAM layout
- **vram-maps/** — VRAM layout
- **rom-maps/** — ROM header/structure
- **timing/** — Cycle timing
- **spc700/** — SPC700 addressing modes
- **official-jargon/** — Official Nintendo terminology
- **65c02-additions/** — 65C02 instructions inherited by 65816
- **inherited-from-6502/** — 6502 compatibility notes
