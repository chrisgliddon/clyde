The **ROM Size** code is stored at address FFD7h:

```
09h....3 ~ 4M Bit
0Ah....5 ~ 8M bit
0Bh....9 ~ 16M Bit
0Ch....17 ~ 32M Bit
0Dh....33 ~ 64M Bit
```
