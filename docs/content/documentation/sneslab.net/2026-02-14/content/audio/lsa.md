**LSA** stands for Source Loop Start Address.\[1] It is 16 bits wide.

Similarly, SA stands for Source Start Address. It is 16 bits wide.

### Reference

1. subparagraph 7.3.1.2 of [page 3-7-11 of Book I](https://archive.org/details/SNESDevManual/book1/page/n177) of the official Super Nintendo development manual
