The **Echo Mixer** combines all the voices which have echo enabled on them (via EON), as well as the [Echo Feedback](/mw/index.php?title=Echo_Feedback&action=edit&redlink=1 "Echo Feedback (page does not exist)"), when the Echo Buffer write enable flag (bit 5 of FLG) is set.
