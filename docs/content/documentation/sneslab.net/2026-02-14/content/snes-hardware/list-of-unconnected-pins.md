According to the jwdonal schematic, several of the ICs on the SNES Motherboard appear to have pins that are not connected. Some of them are:

### S-ENC

- pin 15, VA
- pin 18, PDO
- pin 17, PHA

### S-SMP

- pins 18-34, P40-P57
- pin 17, P5RD

### S-DSP

- pin 79, CK1
- pin 80, CK2
- pin 77, XCK
- pin 40, MCK
- pin 21, DIP
- pin 20, MA15
- pin 1, DKD
- pin 2, MXK
- pins 3-5, MX1-MX3

### Stereo DAC

- pin 5, AGND

### S-WRAM

- pin 55, G

### S-PPU2

- pin 2, /PED
- pin 4, /TOUMEI
- pins 77-89, TST0-TST11

### S-PPU1

- pin 46, VA15

### S-CPU

- pins 19-24, JPIO0-JPIO5
- pins 38-29, JPOUT1-JPOUT2
- pin 70, /DMA
- pin 71, DMA
- pin 80, R/W
- pin 88, /ALCK
- pin 89, /VP
- pin 87, VDA
- pin 86, VPA
- pin 84, XF
- pin 83, MF
- pin 82, /ML

### F411 CIC

- pin 5, NC
- pin 6, CL2
- pin 12, P12
- pin 13, P13
- pin 14, NC
- pin 15, P20
- pin 16, P21
- pin 17, P22

### Hex Inverter

- pin 10, U9E
- pin 12, U9F

### Fully Connected Components

These components appear to have no unconnected pins:

- voltage supervisor
- voltage regulator
- Multi-Out
- RF Modulator
- Front Panel Daughterboard
- Expansion Port
- Cartridge Slot
- both ARAM chips
- both VRAM chips
