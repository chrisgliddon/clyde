There are multiple buses connected to the 5A22:

- Address Bus A, which is what you are probably looking for - this is 24-bits wide because of the 65c816's 24-bit address space
- the 8-bit CPU Data Bus
- Address Bus B is also connected, but this is not usually referred to as the "CPU Bus"
