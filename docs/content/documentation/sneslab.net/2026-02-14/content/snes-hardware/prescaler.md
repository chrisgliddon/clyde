The **Prescaler** is an 8-bit frequency divider on the SPC700. It receives CPUK (2.048 MHz clock) as input and produces an 8 KHz and 64Hz clock as output. A block diagram of the prescalar can be found in the official Super Nintendo development manual. \[1]
