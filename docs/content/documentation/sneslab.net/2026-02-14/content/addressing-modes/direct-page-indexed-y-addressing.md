**Direct Page Indexed Y, Addressing** is supported by two instructions:

- LDX (opcode B6)
- STX (opcode 96)

The effective addresses generated by it are always in bank 0.\[3]

#### Syntax

```
LDX dp, Y
STX dp, Y
```
