**Direct Page Indexed X, Addressing** is supported by:

- ADC (opcode 75)
- AND (opcode 35)
- ASL (opcode 16)
- BIT (opcode 34)
- CMP (opcode D5)
- DEC (opcode D6)
- EOR (opcode 55)
- INC (opcode F6)
- LDA (opcode B5)
- LDY (opcode B4)
- LSR (opcode 56)
- ORA (opcode 15)
- ROL (opcode 36)
- ROR (opcode 76)
- SBC (opcode F5)
- STA (opcode 95)
- STY (opcode 94)
- STZ (opcode 74)

The effective addresses generated by it are always in bank 0.\[3]

#### Syntax

```
LDA dp, X
```
