**Direct Page Indirect Indexed, Y Addressing** is supported by eight instructions.

- ADC (opcode 71)
- AND (opcode 31)
- CMP (opcode D1)
- EOR (opcode 51)
- LDA (opcode B1)
- ORA (opcode 11)
- SBC (opcode F1)
- STA (opcode 91)

#### Syntax

```
LDA (dp), Y
```
