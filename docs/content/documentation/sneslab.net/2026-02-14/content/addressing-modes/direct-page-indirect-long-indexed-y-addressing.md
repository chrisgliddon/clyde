**Direct Page Indirect Long Indexed, Y Addressing** is supported by eight instructions:

- ADC (opcode 77)
- AND (opcode 37)
- CMP (opcode D7)
- EOR (opcode 57)
- LDA (opcode B7)
- ORA (opcode 17)
- SBC (opcode F7)
- STA (opcode 97)

#### Syntax

```
LDA (dp), Y
```
