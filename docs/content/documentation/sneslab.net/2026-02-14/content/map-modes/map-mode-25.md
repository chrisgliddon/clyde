**Map Mode 25** is ExHiROM, used only by the Japanese games:

- *Tales Of Phantasia*
- *Dai Kaiju Monogatari*

The byte at FFD5h is 35h for FastROM and 25h for SlowROM in mode 25.

The board SHVC-LJ3M-01 uses ExHiROM.
