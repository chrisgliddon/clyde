**Map Mode 20** is LoROM. The byte at FFD5h may be 20h for SlowROM or 30h for FastROM in Map Mode 20.

In LoROM, cartridge ROM information is mapped to the upper 32K of each bank.

Use LoROM if you want to be able to access ROM and the system area without switching banks.
